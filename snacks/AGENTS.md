# Project: Snack JS

## Setup
- Apri il file `index.html` nel browser.
- Usa la console del browser per vedere i risultati (`F12` → Console).

## File Structure
- `index.html`: file principale che include gli script.
- Cartella `snacks/`: contiene un file JS per ogni esercizio.
  - `snack1.js` ... `snack9.js`
  - `lavatrice.js`

## Code Style
- Usa `const` e `let` (no `var`).
- Nomi di variabili chiari e descrittivi.
- Commenta i passaggi importanti se la logica non è immediata.

## Agent Rules
- Non modificare `index.html` a meno che non venga richiesto.
- Lavora su un solo file JS alla volta (uno script attivo per volta in `index.html`).
- Quando modifichi il codice, mantieni la logica semplice e leggibile (livello base didattico).
